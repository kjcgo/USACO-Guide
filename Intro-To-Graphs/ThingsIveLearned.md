Livestock Lineup
- Ternary operator:
  ```int temp = t == prev ? s : t;```
  temp will be set to s if t is equal to prev, and t if t is not equal to prev

Milk Factory
- Sink: node in a graph that all other nodes lead to
- In a tree with n-1 directed edges, the sink will be the only node with no outgoing edges (assuming a sink exists)
